# Шпоргалка для начинающих


## Создание репозитрия


Для того, чтобы соать репозиторий небходимо ввести команду: 


```

git init
```


### Создание файлов в репозитории


Для того, чтобы создать файлы в репзитории необходимо использовать команду: 


```

touch
```


Контекст: 


```

touch 'название файла'.'расширение файла'
```


### Добавление файла в репозиторий


После содания файла, необходимо внести его в репозитрий, испольую функцию 


```

git add
```


Контекст: 


```

git add 'файл'
```


или же:


```
git add -all
```


#### Проверка того, что файл внесён в репозиторий


Чтобы убедиться в том, что файл внесён в репозиторий, используйте функцию: 


```

git log
```


##### Уведомление


Перед внесением файла в GitHub, добавьте уведомление, с информацией о том, какие изменения или поправки были обавлены в репозиторий


Контекст: 


```

git commit -m 'текст уведомления'
```


###### Внесение именений в GitHub


После проверки, что все файлы созданы, и что в них необходимые изменения, вносим их в GitHub, с помощью функции: 


```

git push
```
